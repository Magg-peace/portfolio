# Meghana S — Software Developer Portfolio

Welcome to the source code for my personal portfolio website! This repository contains the code for my professional developer portfolio, designed to showcase my skills, projects, and professional experience.

## 🚀 Overview

This portfolio is a single-page, responsive web application built with a focus on clean design, performance, and accessibility. It highlights:

- **About Me**: A brief introduction to my background and expertise.
- **Projects**: A curated list of my recent and most impactful software development projects.
- **Skills**: A summary of my technical skills and the technologies I work with.
- **Experience**: My professional work history.
- **Contact**: Ways to get in touch with me for collaborations or opportunities.

## 🛠️ Built With

- **HTML5**: For semantic structuring of the content.
- **CSS3**: For styling, utilizing custom properties (CSS variables), flexbox, grid, and responsive media queries.
- **Vanilla JavaScript**: (If applicable) for interactive elements and smooth scrolling.
- **Google Fonts**: Inter, Fraunces, and IBM Plex Mono for typography.

## 💻 Getting Started

To run this project locally, simply clone the repository and open the `index.html` file in your web browser. No complex build tools or local servers are required!

```bash
# Clone the repository
git clone https://github.com/your-username/portfolio.git

# Navigate to the directory
cd portfolio

# Open in your default browser (Mac)
open index.html

# Open in your default browser (Windows)
start index.html
```

## ✨ Features

- **Responsive Design**: Adapts beautifully to mobile, tablet, and desktop screens.
- **Dark/Light Theme**: Support for custom theming (can be configured via CSS variables).
- **Fast Loading**: Optimized for performance with minimal dependencies.
