## 🌟 Overview

Welcome to the open-source repository for my developer portfolio! This project serves as my digital resume and a central hub for my professional identity. It is built as a single-page application with a strong focus on:

- **Clean Design Aesthetics**: A modern, readable, and visually appealing user interface.
- **Optimal Performance**: Lightning-fast load times with zero heavy dependencies.
- **Responsive Layout**: Flawless experience across mobile, tablet, and desktop devices.

## 🛠️ Tech Stack

This project was intentionally built using core web technologies to demonstrate fundamental web development skills without relying on heavy frameworks:

- **HTML5**: Semantic and accessible markup.
- **CSS3**: Advanced styling leveraging custom properties (variables), Flexbox, CSS Grid, and responsive media queries.
- **Vanilla JavaScript**: Lightweight DOM manipulation for interactive elements.
- **Typography**: Utilizing `Inter`, `Fraunces`, and `IBM Plex Mono` via Google Fonts.
- **Deployment**: Hosted on [Vercel](https://vercel.com).

## 🚀 Getting Started Locally

If you'd like to run this project locally to explore the code or use it as inspiration, follow these steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/Magg-peace/portfolio.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd portfolio
   ```

3. **Open the project**
   - Simply double-click the `index.html` file to open it in your default web browser, or drag and drop it into a browser window. No build tools required!
