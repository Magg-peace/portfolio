Welcome to the open-source repository for my personal developer portfolio! This project is designed to professionally showcase my software development skills, recent projects, and professional background.

## 🌟 Overview

This portfolio is a single-page, fully responsive web application built with a strong focus on clean design aesthetics, optimal performance, and web accessibility. It serves as a central hub for my professional identity.

### Key Sections:
- **About Me**: A professional summary of my background, expertise, and passion for software development.
- **Projects**: A curated showcase of my most impactful work, complete with descriptions and technologies used.
- **Skills**: A comprehensive overview of my technical stack and core competencies.
- **Experience**: My professional trajectory and work history.

## 🛠️ Tech Stack

This project was built focusing on core web technologies to ensure maximum performance and minimal overhead:

- **HTML5**: Semantic and accessible markup.
- **CSS3**: Advanced styling leveraging custom properties (variables), Flexbox, CSS Grid, and responsive media queries.
- **Vanilla JavaScript**: Lightweight DOM manipulation and interactive elements.
- **Typography**: Utilizing `Inter`, `Fraunces`, and `IBM Plex Mono` via Google Fonts.
- **Deployment**: Hosted statically on [Vercel](https://vercel.com).

## 🚀 Getting Started

If you'd like to run this project locally to explore the code or use it as inspiration, follow these steps:

### Prerequisites
No special tools are required! All you need is a web browser and a code editor (like VS Code).

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Magg-peace/portfolio.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd portfolio
   ```

3. **Open the project**
   - On **Mac**: `open index.html`
   - On **Windows**: `start index.html`
   - Or simply drag and drop the `index.html` file into your preferred web browser.

## 🎨 Features & Highlights

- **Fully Responsive Design**: Ensures a seamless experience across mobile phones, tablets, and large desktop monitors.
- **Custom Theming Architecture**: Built with a scalable CSS variable system allowing for easy implementation of Light/Dark modes.
- **Zero-Dependency Architecture**: Built without heavy frameworks to guarantee lightning-fast load times and a perfect Lighthouse score.
